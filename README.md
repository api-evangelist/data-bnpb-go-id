# Satu Data Bencana Indonesia (BNPB) (data-bnpb-go-id)

Satu Data Bencana Indonesia (BNPB) is a government agency open-data portal for Indonesia running **CKAN**, exposing a standardized catalog API over ~2,830 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-bnpb-go-id/refs/heads/main/apis.yml)
